# 🧠 KURO – Your Second Brain, Reimagined

> Think. Write. Organize. All in one place.

KURO is a modern, minimal note-taking and workspace app inspired by tools like Notion—designed to help you capture ideas, manage content, and build your personal knowledge system effortlessly.

---

## ✨ Features

- 📝 **Rich Text Editor**  
  Create and edit notes with a clean, distraction-free interface

- 📂 **Workspace Management**  
  Organize your pages, notes, and ideas in a structured way

- 🔍 **Quick Navigation**  
  Sidebar access for fast switching between pages

- 📌 **Pages System**  
  Create, edit, and manage multiple documents

- 🗑️ **Trash & Recovery**  
  Manage deleted notes easily

- ⭐ **Favorites (UI Ready)**  
  Mark important pages for quick access *(extendable feature)*

- 🚀 **Demo Mode Access**  
  Instantly explore the app without signup
